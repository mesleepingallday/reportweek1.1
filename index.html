1. An introduction

Nếu HTML là xương sống của web pages thì JavaScript tạo ra chuyển động, và Js được chạy đơn giản bằng văn bản thuần túy, trên bất kỳ thiết bị có JavaScript engine

JavaScript trong trình duyệt có thể thực hiện mọi thứ liên quan đến thao tác trang web, tương tác với người dùng và webserver:

  • Thêm HTML mới vào trang, thay đổi nội dung hiện có, sửa đổi kiểu.
  • Tương tác với hành động của người dùng, chạy khi nhấp chuột, di chuyển con trỏ, nhấn phím.
  • Gửi yêu cầu qua mạng đến các máy chủ từ xa, download và upload tệp
  • Nhận và đặt cookie, đặt câu hỏi cho khách truy cập, hiển thị tin nhắn.
  • Ghi nhớ dữ liệu ở phía máy khách (“bộ nhớ cục bộ”).
Cookie là một tập tin nhỏ được lưu trữ trên máy tính của người dùng bởi một trang web. Nó có thể được sử dụng để nhắc nhở người dùng về những thứ đã thực hiện trên trang web, như đăng nhập hoặc các mục đã xem.

Khả năng của JavaScript bị giới hạn để bảo vệ thông tin dữ liệu của người dùng:

  • Không đọc/ghi tệp tin nào vào hard disk, sao chép và chạy chương trình.
  • Các tab khác nhau thường không biết về nhau. Chỉ biết trong trường hợp cả hai đồng ý trao đổi dữ liệu và chứa Js code đặc biệt
  • JavaScript có thể dễ dàng giao tiếp qua mạng với máy chủ nơi xuất phát của trang hiện tại. Nhưng khả năng nhận dữ liệu từ các trang web/tên miền khác bị tê liệt

Dựa trên nhiều mục đích sử dụng khác nhau của mỗi người, Transpiled languages được tạo ra

Transpiled languages là những ngôn ngữ lập trình chuyển đổi từ mã nguồn nguyên thủy sang mã nguồn đích khác. Chúng thường được dùng để tối ưu hóa hiệu suất hoặc thay đổi ngôn ngữ lập trình, ví dụ như chuyển đổi từ mã nguồn TypeScript sang mã nguồn JavaScript. Chuyển đổi này được thực hiện bởi một trình dịch viên, một chương trình máy tính hoặc các công cụ trực tuyến.

Modify-in-place

cho phép bạn thay đổi giá trị của một đối tượng hoặc mảng mà không cần tạo một bản sao mới của nó.

Thường dùng với toán tử += , *= ,…

let n = 2;
n += 5; // now n = 7 (same as n = n + 5)
n *= 2; // now n = 14 (same as n = n * 2)

alert( n ); // 14

Increment/decrement

  • ++ tăng giá trị thêm 1 đơn vị
let a = 1;
a++;
alert(a); // a=2
  • -- giảm giá trị đi 1 đơn vị
let a = 1;
a--;
alert(a); // a=0

Sự khác biệt giữa ++value và value++ là chúng tương tác khác nhau với biến value khi sử dụng trong một câu lệnh.

  • ++value (toán tử tăng trước) sẽ tăng giá trị của biến value trước khi nó được sử dụng trong một câu lệnh.
let x = 5;
let y = ++x;
alert(x); // x = 6
alert(y); // y = 6
  • value++ (toán tử tăng sau) sẽ sử dụng giá trị của biến value trước khi nó được tăng.
let x = 5;
let y = x++;
alert(x); // x = 6
alert(y); // y = 5

Theo thứ tự ưu tiên thì ++ >+,-,*,/

let x = 1;
alert(2 * ++x); // 4
alert(2 * x++); // 2

Comparisons

toán tử so sánh là một loại toán tử dùng để so sánh hai giá trị và trả về một giá trị true hoặc false

  • Lớn, bé hơn: a > b, a < b.
  • Lớn hơn, bé hơn hoặc bằng: a >= b, a <= b.
  • Bằng nhau ( so sánh): a == b
  • Khác nhau:  a != b.
alert( 2 > 1 );  // true (correct)
alert( 2 == 1 ); // false (wrong)
alert( 2 != 1 ); // true (correct)

có thể gán giá trị logic cho biến

let result = 5 > 4; // gan ket qua cua phep so sanh
alert( result ); // true

String comparison

quá trình so sánh hai chuỗi và trả về một giá trị true hoặc false.

alert("hello" == "hello");  // true

Thuật toán: so sánh từng kí tự của hai chuỗi với nhau từ trái qua phải, nếu kí tự nào lớn hơn thì chuỗi lớn hơn. Nếu so sánh hết chiều dài chuỗi vẫn bằng nhau thì 2 chuỗi bằng nhau. Chuỗi nào có dài hơn thì lớn hơn

Comparison of different types

khi so sánh hai giá trị có kiểu dữ liệu khác nhau, một số quy tắc chuyển đổi kiểu dữ liệu được áp dụng để cho phép so sánh

  • Nếu một trong hai giá trị là một số, thì giá trị còn lại sẽ được ép về số trước khi so sánh.

  • Nếu một trong hai giá trị là một chuỗi, thì giá trị còn lại sẽ được ép về chuỗi trước khi so sánh
alert(5 == "5"); //true
  • Nếu một trong hai giá trị là một đối tượng, thì giá trị còn lại sẽ được ép về đối tượng trước khi so sánh (nếu không sẽ trả về false)

Strict equality (===)

một loại so sánh giống như equality (==), nhưng nó so sánh giá trị và kiểu dữ liệu của hai biến. Nếu giá trị và kiểu dữ liệu của hai biến khớp nhau, nó sẽ trả về true, ngược lại sẽ trả về false.

let x = 5;
let y = "5";
alert(x === y);  // false

Logical operators

Trong JavaScript, có 3 toán tử logic: AND (&&), OR (||), NOT (!).

  • AND (&&): Sẽ trả về true nếu cả hai toán hạng đều là true, ngược lại sẽ trả về false.
  • OR (||): Sẽ trả về true nếu một trong hai toán hạng là true, ngược lại sẽ trả về false.
  • NOT (!): Sẽ đảo ngược giá trị logic của toán hạng.
alert(true && true);   // true
alert(true && false);  // false
alert(false && false); // false

alert(true || true);   // true
alert(true || false);  // true
alert(false || false); // false

alert(!true);   // false
alert(!false);  // true

Nullish coalescing operator '??'

Toán tử nullish coalescing (??) cho phép xác định giá trị mặc định cho một biến hoặc thuộc tính nếu nó là null hoặc undefined. Toán tử này sẽ trả về giá trị bên trái nếu nó khác null hoặc undefined, ngược lại nó sẽ trả về giá trị bên phải.

let a = null;
let b = undefined;
let c = "Hello";

alert(a ?? "default value");  // "default value"
alert(b ?? "default value");  // "default value"
alert(c ?? "default value");  // "Hello"

Loops: while and for

  • Vòng lặp while: Sẽ lặp lại một block code cho đến khi điều kiện trong ngoặc tròn () là true.
let i = 0;
while (i < 5) {
  alert(i);
  i++;
}
  • Vòng lặp for: Sẽ lặp lại một block code cho đến khi điều kiện trong ngoặc tròn () là false. Trong vòng lặp for có 3 phần chính: khởi tạo, điều kiện và sự tăng giảm của biến đếm.
for (let i = 0; i < 5; i++) {
	 alert(i);
}

Cả hai vòng lặp while và for có thể dùng với các hàm break và continue để dừng hoặc bỏ qua một vòng lặp nào đó.

  • sử dụng hàm break để dừng vòng lặp ngay lập tức, khi gặp điều kiện nào đó trong vòng lặp
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break; //vòng lặp sẽ dừng lại sau khi in ra số 5
  }
  alert(i);
}
  • hàm continue sẽ bỏ qua lần lặp hiện tại và tiếp tục vòng lặp kế tiếp
for (let i = 0; i < 10; i++) {
  if (i % 2 === 0) {
    continue; //vòng lặp sẽ in ra các số lẻ từ 0 đến 9, bỏ qua số chẵn
  }
  console.log(i);
}

The "switch" statement

switch statement cho phép bạn so sánh một biến hoặc giá trị với nhiều giá trị khác nhau và thực hiện một block code tương ứng

switch (expression) {
  case value1:
    // code to be executed if expression === value1
    break;
  case value2:
    // code to be executed if expression === value2
    break;
  ...
  default:
    // code to be executed if expression is not matched with any case
}

E.g

let day = "Monday";
switch (day) {
  case "Monday":
    alert("Today is Monday");
    break;
  case "Tuesday":
    alert("Today is Tuesday");
    break;
  case "Wednesday":
    alert("Today is Wednesday");
    break;
  default:
    alert("Today is another day");
}
//giá trị của biến "day" là "Monday", nên sẽ in ra "Today is Monday".

Cần sử dụng break để dừng vòng lặp sau khi thực hiện block code tương ứng, nếu không sẽ tiếp tục chạy các block code tiếp theo cho đến khi gặp break hoặc kết thúc switch.

Trong một số trường hợp nếu muốn thực hiện nhiều lệnh cho nhiều giá trị giống nhau có thể sử dụng fall-through

let x = 2;
switch (x) {
  case 2:
  case 3:
    alert("x is 2 or 3");
    break;
  default:
    alert("x is something else");
}
